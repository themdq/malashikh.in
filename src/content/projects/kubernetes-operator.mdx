## Overview

This project implements a Kubernetes operator that automates the provisioning and management of PostgreSQL databases within a Kubernetes cluster.

## Features

- Automated database creation and deletion
- Credential management via Kubernetes Secrets
- Backup scheduling and restoration
- Connection pooling configuration
- High availability setup with replicas

## Technical Details

The operator is built using the Operator SDK and follows Kubernetes best practices for custom resource definitions (CRDs) and controller patterns.
