## Overview

An end-to-end observability platform that provides metrics, logs, and traces for distributed systems running on Kubernetes.

## Components

- **Metrics**: Prometheus with custom recording rules
- **Logs**: Loki for log aggregation
- **Traces**: OpenTelemetry with Jaeger backend
- **Visualization**: Grafana dashboards

## Features

- Auto-discovery of services
- Pre-built dashboards for common workloads
- Alerting with PagerDuty integration
- Long-term storage with Thanos
