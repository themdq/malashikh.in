## Overview

A modular Terraform-based infrastructure platform that enables consistent deployments across AWS and GCP with automated GitOps workflows.

## Features

- Reusable Terraform modules for common patterns
- Multi-cloud support (AWS, GCP)
- Automated PR reviews for infrastructure changes
- Cost estimation before apply
- State management with remote backends

## Architecture

The platform uses a hub-and-spoke model with shared modules and environment-specific configurations, integrated with ArgoCD for Kubernetes deployments.
